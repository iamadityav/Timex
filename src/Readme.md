# React Native Timer App

A customizable timer app built with React Native, featuring:

- **Timer Management**: Start, pause, and reset timers
- **Categories**: Group timers by categories
- **Progress Visualization**: Simple progress bar for each timer
- **Bulk Actions**: Start, pause, and reset all timers in a category
- **History**: Log of completed timers

## Features

- **Customizable Timers**: Create timers with specific durations and categories
- **Real-time Progress**: Visualize remaining time as a progress bar
- **State Management**: Uses React hooks for efficient state management
- **AsyncStorage**: Persists data locally using AsyncStorage

## Installation

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `npx react-native run-android` or `npx react-native run-ios` to start the app
